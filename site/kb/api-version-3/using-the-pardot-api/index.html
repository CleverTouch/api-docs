API (version 3) - Using the Pardot API

The Pardot API handles a variety of requests for many of the objects available through the Pardot user interface. Most requests to the API use the following standardized format. All requests must use either HTTP GET or POST. Although GET requests are secure due to the use of SSL, we recommend using POST, with sensitive or lengthy parameter values being part of the POST message body. Developers are responsible for issuing requests with the following components:

`POST: https://pi.pardot.com/api/
